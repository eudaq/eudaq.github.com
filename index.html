EUDAQ

a generic data acquisition framework

Basic Installation Instructions:

In order to use and run EUDAQ, we recommend to use a stable release!

In order to get the latest release version (here e.g. v1.6.0), just follow these steps:

git clone https://github.com/eudaq/eudaq.git eudaq
git tag
List you all releases, with the latest as last one. 
git checkout v1.6.0
cd eudaq/build
cmake ..
make install

Detailed Installation Instructions

Please have a look at the README.md file included in the source code, the user's manual (see below), or the EUDAQ wiki pages.

User's Manual

You can download the user's manual or generate the manual from the LaTeX sources included in EUDAQ:

cd build
cmake -DBUILD_manual=ON ..
make pdf

The manual can then be found in ./doc/manual/EUDAQUserManual.pdf

Doxygen Source Code Documentation

The doxygen documentation of the EUDAQ source code can be found online or generated from the source code itself:

cd build
cmake ..
make doxygen

The resulting HTML files are stored in ../doc/doxygen/html.

Release Notes

You can find the release notes in the README.release_notes file in the repository or online.

Support and Bug Tracker

Having trouble with EUDAQ or found a bug? Please report it on the EUDAQ issue tracker and we'll try to sort it out!
