EUDAQ

a generic data acquisition framework

Basic Installation Instructions:

To get the current development version, just follow these steps:

git clone https://github.com/eudaq/eudaq.git eudaq
cd eudaq/build
cmake ..
make install

For production environments we recommend to download a stable release!
